# 4-bit ALU using Verilog

## Project Description
This project implements a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL. It performs arithmetic and logical operations like addition, subtraction, AND, OR, XOR.

## Tools Used
- Verilog HDL  
- EDA Playground / Simulation Tool  

## Output
Waveform simulation verifies correct functionality of ALU operations.
